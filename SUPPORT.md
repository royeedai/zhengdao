# Support

如果你在使用证道时遇到问题，建议按下面顺序处理。

## 1. 先看哪些问题适合提 issue

适合提 issue：

- 可稳定复现的 bug
- 安装失败或升级失败
- 导入导出异常
- AI / 云同步配置异常
- 明确的功能建议

不适合直接提公开 issue：

- 安全漏洞
- 包含敏感数据、私密文稿、令牌或隐私信息的问题

安全问题请参考 [SECURITY.md](./SECURITY.md)。

## 2. 提交 bug 前请准备这些信息

- 应用版本
- 操作系统和架构
- 复现步骤
- 预期结果与实际结果
- 错误截图、日志或录屏

## 3. 使用哪种模板

- 稳定复现问题：Bug Report
- 新能力建议：Feature Request

## 4. 关于更新

- 正式版会通过 GitHub Releases 提供安装包
- 已安装正式版会在后台检查并下载新版本
- 下载完成后标题栏会出现 `更新`
- macOS 如果未签名 / 未公证，系统可能拦截安装或自动更新
