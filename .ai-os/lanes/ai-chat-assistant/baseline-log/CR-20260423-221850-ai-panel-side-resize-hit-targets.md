# 变更记录：AI 助手面板补左右边缘缩放热区

- **基线 ID**：CR-20260423-221850-ai-panel-side-resize-hit-targets
- **时间**：2026-04-23 22:18
- **类型**：bugfix / interaction
- **影响级别**：P1

## 变更原因

用户反馈 AI 创作助手当前左右拉宽不方便，两个角点手柄经常拉不动左右。现有实现只有 16px 角点热区，且左上角靠近标题栏拖拽区，横向调宽时容易点不中或被拖动面板手势抢走。

## 本次变更内容

1. 保留左上角与右下角两个角点缩放手柄。
2. 新增左边缘和右边缘透明拖拽热区，支持直接横向调整宽度。
3. 左边缘缩放固定右边界，右边缘缩放固定左边界。
4. 补左右边缘缩放布局单测，继续约束最小宽度、最大宽度和视口边界。

## 影响分析

| 维度 | 是否受影响 | 说明 |
|------|------------|------|
| DESIGN | 是 | AI Dock 缩放从角点扩展为边缘热区 |
| spec | 是 | AC-AI-010 的拖拽缩放需包含左右边缘横向拉伸 |
| tests | 是 | 补左右边缘缩放 guard |
| release | 否 | 不涉及发布动作 |
| memory | 否 | 当前 lane 局部交互修复 |

## 风险

- 左右边缘热区不能覆盖面板内部按钮、输入框的正常点击区域。
- 左边缘缩放需保持右边界稳定，避免面板横向跳动。
