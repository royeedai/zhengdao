## 确认决策

- 更新源采用 `GitHub Releases`
- Electron 客户端采用 `electron-updater`
- 按钮语义采用“后台自动检查 + 自动下载；仅下载完成后显示 `更新`”
- 点击 `更新` 后不弹确认，先 flush 当前未保存编辑内容，再触发重启安装
- Windows 在线更新作为可用目标直接支持
- macOS 代码侧接入在线更新，但真正上线仍依赖签名 + notarization

## 新增风险 / blocker

- macOS 自动更新正式上线仍受签名与 notarization 阻塞
- 当前资源目录仍缺正式 `icon.icns` / `icon.ico`
- GitHub Releases 发布能力依赖 `GH_TOKEN` 与签名相关 secrets
