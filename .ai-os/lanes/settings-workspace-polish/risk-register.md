# 风险登记

| 风险 | 影响 | 缓解 |
|---|---|---|
| schema / migration 改动影响已有作品配置读取 | 中 | 先补 migration 测试，再做 renderer 接线 |
| 设置分层重构导致入口迁移后能力不可达 | 中 | 逐项对照系统/作品设置 IA，补 code review 清单 |
| 顶栏双击与交互按钮冲突 | 中 | 只在非交互背景区域触发，按钮区保持 `no-drag` |
| 创世沙盘拖拽改造引入点击/拖拽误判 | 中 | 使用明确阈值与 pointer drag 状态机分离 |
| 旧样式页面统一时误伤业务逻辑 | 中 | 限定为样式和 token 替换，不改数据流 |
